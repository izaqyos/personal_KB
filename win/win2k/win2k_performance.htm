Introduction:

If you are like me you have a love/hate relationship with Windows 2000.  Lets face it you can’t get a singe Electronic Arts Game to run on it, some of the best hardware still has marginal driver support for it, and Win2k takes a lot of cycles just to run in the background.  Well hopefully this will take some of the hate out of you.  So get ready, say a prayer to your favorite deity and get ready to make your Win2k install scream.

 

 

Optimize your install of Win2k:

There are a few things that getting right before you install will make a world of difference.

 

Use Fat:  There is a lot of debate about NTFS vs. Fat32.  Unless you are worried that your room mate is going to read your love letters to Eminem don’t bother with NTFS it is slower and God help you if you ever have to clean up a problem from dos, or Win9x.  I’ll post a quick article on how to make an unbeatable Password for Win2k later I promise.

Dual Boot:  A Windows 98 install is very important to have if you are running Windows 2000 as your primary operating system.  (Win95 with USB will work too)  For only about 150 Megs of headache you insure that you will still be able to play all of your favorite games.  But oddly I have found that there are games that will only install under Win9x but will run great under 2000.  This is because they detect Win2k as WinNT.

 

Service Pack, and Patch the h*!! out of it:  I often tell people not to run a MS product until the first service pack is available, as MS has a tendency to release products in a practically beta state.  Since the dawn of Windows Update that is less of an issue.  There are currently about 150 Megs of updates for Win2k, including three compatibility patches to make Win2k friendlier to old Win9x and Win3.x.

 

Swap File:  Quick rules on swap files

 

  • Swap files work best on a separate drive or partition of about 10% larger than the swap file.
  • Swap file should bring you up to at least 384 of total memory (ram + Swap)
  • Swap file should not take you to beyond 1 gig of total memory (unless you are running specific software requiring it)  (you aren’t or you would be able to name one)
  • Swap file should be set to a specified size with both the Minimum and Maximum set to the same size.

Cut Down Your Network Traffic: I sat down to a friend’s PC, he has one NIC card attached to a cable modem, but had TCP-IP, IPX, NET BUI, Apple Talk, And Internet connection sharing.  Use only the protocols you need.  If your computer is only attached to a cable modem then this is TCP-IP.

 

Get the most out of your Hard Drive: As you may know, DMA (Direct Memory Access) is a system that allows devices in your computer to transfer data directly to and from RAM without having to use the CPU as an intermediary.  This system is a boost to overall system performance not only because the devices can respond more quickly but it also frees up the CPU to perform other mission critical tasks.  UDMA (Ultra Direct Memory Access) is an enhancement of the DMA system that makes this system even more effective by boosting the data transfer rates.

Most computer systems capable of running Windows 2000 are designed to use either DMA or UDMA to communicate with the CD-ROM and hard drive.  However, in order for the system to operate correctly, the operating system must be configured to enable DMA/UDMA mode.

When you install Windows 2000, DMA or UDMA should be enabled for your hard drive.  However, I've discovered that is not always the case.  As such, you need to check the Windows 2000 settings and make sure that your system is getting the performance boost that DMA provides.

To begin, right-click on My Computer to bring up the System Properties dialog box.  Then select the Hardware tab and click the Device Manager button.  When you see the Device Manager window locate and open the IDE ATA/ATAPI controllers branch.  Then, double-click the Primary IDE Channel.  When you see the Primary IDE Channel Properties dialog box, select the Advanced Settings tab.

Now, check that Current Transfer Mode setting to see if it reads DMA or Ultra DMA Mode.  If it does, you're all set.

If it doesn't, select Auto Detection from the Device Type drop down list and click OK.  Then, close every open window and restart your system.  Then, return to Device Manager and check again.

If the Current Transfer Mode setting still isn't set to DMA or Ultra DMA mode, select DMA if available from the Transfer Mode drop down list and click OK.  Then, close every open window and restart your system.  Then, return to Device Manager and check again.

If at this point the Current Transfer Mode setting still isn't set to DMA or Ultra DMA mode, you may need to access your system BIOS and check to make sure the DMA mode setting is enabled there.  If it is, try disabling it and see if that has any effect.  You may also want to check the computer manufacturer's Web site for more information.

 

 

 

 

Turn off the bells and whistles:  Things to turn off and there are a lot of them.

 

 

Effects:  While they were fun for the first 5 minutes the Start menu effects are annoying the second week.  Show Window Contents while Dragging is only useful if you move between virtual desktops a lot.

 

 

 

Screensaver:  Screen savers only save your monitor if you have an old Amber Glow, or Mono Chrome monitor.  Screensavers always crop up at the worst times, and suck up a lot of resources.

 

 

Power saving:  Power saving is almost dangerous.  Your hard drive was meant for continuous duty, so unless you have major heat issues, or are on a laptop letting your hard drives spin down is not needed.

 

I use power saving on my monitor because I like the savings on my electric bill, and I think my monitor get a bit hot after about 18 hours of use.

 

 

 

 

Backgrounds:  Backgrounds eat a lot of memory and turning them off can make a lot more difference than you might think

 

 

 

Unless you are running photo software stick to 16bit color:

 

 

Turn off Active Desktop:  Just like in Windows 98 the active desktop will grind your hard drive, slow down your PC and generally cause problems.

 

 

 

Disable Web Content in Windows Explorer:

While it doesn’t make much difference when you are playing Quake, this change will make Explore load faster and browse faster.

 

From within Explorer choose Tools -> Folder Options, then under the General tab choose “Use Windows classic folders”

 

 

 

Turn off the Task Manager:  Task Manager is really only a good thing if you are using the backup software that comes with Win2k, or if you are religious about scheduling your Defrags and Scan Disk.  From the Administration Tools run the Services Applet, then choose the Task scheduler and disable it.

 

 

 

 

 

Turn off all the other things you probably don’t need:  Windows 2000 is really meant for corporate desktops, and as such it has a lot of security and network features the majority of home users won’t need.

 

Disclaimer:  You can really muck up your PC if you turn to many things off.  So if you are faint of heart, skip to the next section.

See the Big Picture:  The Personalized Menus feature is at the top of the list.  It hides some items on the Programs menu, based on your usage patterns; the practical effect is that you spend more time trying to find the hidden items.

Choose Settings, Taskbar & Start Menu, and clear the Use Personalized Menus box.  Then click the Advanced tab (shown below), and check or uncheck boxes to decide which items you want on the Start menu.

 

Silence the beast:  I was amazed at how much memory the sounds in Win2k eat up.  (Some of the longer ones used nearly a Meg).  So I disable them.  This is a simple task.  Simply select the “Sounds and Multimedia” from control panel and set the "Schemes" setting to “No Sounds”.

 

 

 

 

 

Cleanliness is next to Godliness:  Keep your files organized.  This is a lead in to the next section, but keeping your files on a separate drive or partition makes a huge difference in keeping your hard drive(s) from getting fragmented.  It also makes saving them to a Back up easier.

Defrag your Hard Disks Often:  You know that Grinding noise that your machine makes when you try and get in to American McGee’s Alice?  That is your hard drive choking on the HUGE size of the animations and Textures.  That Whirring noise as Mech Warrior 4 tries to play a Cut scene that is your hard drive trying to cache the CD.  Want to make both those, faster, smoother, and extend the life of your Hard Disk?  DEFRAG!  Yes I understand that since you haven’t done it since the dawn of time that it takes 18 hours, but you can sleep, go to the park, or go see your neglected Girlfriend, Spouse, or Goldfish.

Ghost:  Norton Ghost rocks so hard.  It costs a little but will save you so much trouble.  I loathe installs of my operating system and E-mail configuration, and Office 2000 settings, and all the rest of that stuff.  So I use Ghost.  Ghost will let you make a complete image of your hard drive or partition.  You can then destroy the hell out of your install and be certain to put it back…  Make sure you don’t have any problems before you put your faith in Ghost, on more than one occasion my Tech’s have had a virus on the Ghost, and then tried to cure their Virus Problems with the Ghost (Doh!)  If you defrag first a Ghost will be smaller, and take less time to make.

Help for when you Blow the whole thing up:  People gripe about how hard it is to get answers to Windows Problems.  I find 99% of my problems have an answer in the Microsoft Knowledge Base.  Don’t be afraid to use it.

 

 

 

Conclusion:

Windows 2000 brings a lot of very worthwhile features to the table, so don’t give up on just because it is a bit sluggish right out of the box.  After you’ve made all of these improvements you will be impressed at how snappy MS’s Bloatware can be.
