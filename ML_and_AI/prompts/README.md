# Prompt Library

A curated collection of high-quality, proven-to-work prompts for AI image generation, text generation, and creative workflows.

Every prompt in this repository has been tested and refined for consistent, high-quality results.

## Structure

```
prompts/
├── README.md
└── nano_banana_prompts/       # Visual / image-generation prompts
    ├── detailed-object-schematics.md
    ├── glass-sculpture.md
    └── youtube-to-infographic.md
```

## Categories

### Nano Banana Prompts

Highly detailed, visually rich image-generation prompts designed for tools like Midjourney, DALL-E, and similar platforms.

| Prompt | Description | Best For |
|--------|-------------|----------|
| [Detailed Object Schematics](nano_banana_prompts/detailed-object-schematics.md) | Technical infographic teardown of any object — isometric 3D, labeled components, color-coded systems | Product teardowns, educational content, technical illustration |
| [Glass Sculpture](nano_banana_prompts/glass-sculpture.md) | Hyper-realistic glass sculpture with light refraction and caustics — fully parameterized | Art prints, social media, creative portfolios |
| [YouTube to Infographic](nano_banana_prompts/youtube-to-infographic.md) | Two-prompt system (Gemini + YouTube) — summarize any video then visualize as a whiteboard sketch note | Study guides, conference recaps, social content, cheat sheets |

## Usage

Each prompt file contains:

- **The prompt template** with placeholders marked in `[BRACKETS]` or `{{curly braces}}`
- **Parameter options** where applicable
- **Tips** for best results

Copy the template, replace the placeholders with your subject, and paste into your image generation tool.

## Contributing

When adding a new prompt:

1. Test it at least 3 times with different subjects to confirm consistency
2. Place it in the appropriate category folder
3. Include parameter options and usage tips
4. Update this README's table
